# Work Scripts

A collection of scripts related to my daily work as a software engineer.

## Project Overview

| Project | Description | 
| :--- | :--- |
| `dotnet/flaky-test-analyzer.sh` |  Executes `dotnet test` for a specified number of runs to identify flaky tests. Tracks execution time, provides a live ETA, and outputs a detailed failure rate and flaky test breakdown. |
